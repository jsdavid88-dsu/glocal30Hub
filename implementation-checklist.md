# 글로컬30 R&D Hub v3 — 페이지별 구현 체크리스트

이 문서는 실제 개발 순서를 잡고, 화면별 기능 누락을 막기 위한 구현 체크리스트다.
각 페이지에서 필요한 데이터, 주요 기능, 권한 포인트를 함께 정리한다.

## 15. QA 체크 포인트

- [ ] 프로젝트 밖 데이터가 외부업체에 노출되지 않는가
- [ ] advisor/private/internal/project 권한이 정확한가
- [ ] 교수는 자기 지도학생 데이터만 보는가
- [ ] 데일리 블록 권한이 피드/상세/검색에서 모두 일관적인가
- [ ] 태스크 공동할당이 정상 동작하는가
- [ ] 일정 visibility가 제대로 반영되는가
